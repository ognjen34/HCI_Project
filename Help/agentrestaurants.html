Agent Restaurants

Viewing Restaurants

  1. 1.

    After navigating to the Agent Restaurants page, you will see a list of restaurants available.

    The restaurant cards on the Agent Restaurants page will display the following information:

    • Restaurant Name
    • Address
    • City
    • Cuisine Type
    • Rating in stars
  2. 2.

    To add a new restaurant, click on the Add button.

    This will open an empty form for adding a new restaurant.

  3. 3.

    To edit a restaurant, click on the Edit button associated with the restaurant.

    This will open a form for editing the restaurant that the edit button was clicked on.

  4. 4.

    To delete a restaurant, click on the Delete button associated with the restaurant.

    This will open a confirmation dialog warning you about the consequences of deletion.

    THIS ACTION IS IRREVERSIBLE! Once a restaurant is deleted, it cannot be undone!
