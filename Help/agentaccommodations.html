Agent Accommodations

Viewing Accommodations

  1. 1.

    After navigating to the Agent Accommodations page, you will see a list of accommodations available.

    The accommodation cards on the Agent Accommodations page will display the following information:

    • Accommodation Name
    • Location
    • Price per Day
    • Beds
  2. 2.

    To add a new accommodation, click on the Add button.

    This will open an empty form for adding a new accommodation.

  3. 3.

    To edit an accommodation, click on the Edit button associated with the accommodation.

    This will open a form for editing the accommodation that the edit button was clicked on.

  4. 4.

    To delete an accommodation, click on the Delete button associated with the accommodation.

    This will open a confirmation dialog warning you about the consequences of deletion.

    THIS ACTION IS IRREVERSIBLE! Once an accommodation is deleted, it cannot be undone!
