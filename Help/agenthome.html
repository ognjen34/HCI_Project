Agent Home

Viewing Trips

  1. 1.

    After successful login, you will be redirected to the Agent Home page.

    On the Agent Home page, you will see a list of trips available for booking.

    The trip cards on the Agent Home page will display the following information:

    • Trip Name
    • Destination
    • Start Date
    • End Date
    • Price
    • Total sold
    • Total earned
    • Active
  2. 2.

    To see details about past ordered trips and statistics of sold attractions and restaurants in them, click on the Details button

    This will open an advanced info window with a map.

  3. 3.

    To add a new trip, click on the Add button

    This will open an empty form for adding a new trip.

  4. 4.

    To edit a trip, click on the Edit button associated with the trip.

    This will open a form for editing the trip which edit button was clicked on.

  5. 5.

    To delete a trip, click on the Delete button associated with the trip.

    This will open a confirmation dialog warning you about the consequences of deletion.

    THIS ACTION IS IRREVERSIBLE! once a trip is deleted it cannot be undone!
