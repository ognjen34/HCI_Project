Registration Form

Registering

  1. 1.

    There are several inputs to fill out:

    • Name: Enter your first name.
    • Surname: Enter your last name.
    • Username: Enter a valid email address. This will be your username.
    • Password: Enter a password with a minimum length of 8 characters.
    • Confirm Password: Re-enter your password to confirm it.
  2. 2.

    Fill in all the required fields with the appropriate information.

    If any of the fields are left empty, an error message will be displayed indicating which field needs to be filled in.

    If the Name field is left empty, the error message "Fill in the Name field" will be displayed.

    If the Surname field is left empty, the error message "Fill in the Surname field" will be displayed.

    If the Username field is left empty, the error message "Fill in the Username field" will be displayed.

    If the Password field is left empty, the error message "Fill in the Password field" will be displayed.

    If the Confirm Password field is left empty, the error message "Fill in the Confirm Password field" will be displayed.

  3. 3.

    Ensure that the Name field does not exceed 30 characters.

    If the name entered exceeds the character limit, the error message "Name cannot exceed 30 characters!" will be displayed.

  4. 4.

    Ensure that the Surname field does not exceed 30 characters.

    If the surname entered exceeds the character limit, the error message "Surname cannot exceed 30 characters!" will be displayed.

  5. 5.

    Ensure that the Username field is in email format.

    If an invalid email address is entered, the error message "Username needs to be in email format!" will be displayed.

    The correct email format is: example@gmail.com

  6. 6.

    Ensure that the Password field is at least 8 characters long.

    If the password length is less than 8 characters, the error message "Password must be at least 8 characters long!" will be displayed.

  7. 7.

    Ensure that the entered Password and Confirm Password match.

    If the passwords do not match, the error message "Password and Confirm Password do not match!" will be displayed.

  8. 8.

    Check if a user with the entered email already exists in the system.

    If a user with the same email already exists, the error message "User with this email already exists!" will be displayed.

  9. 9.

    If all the validations pass successfully, the registration will be considered successful.

    A success message, "Registration successful!", will be displayed, and the user will be added to the system.

    You will then be redirected to the login page where you can enter your credentials to log into the application.
