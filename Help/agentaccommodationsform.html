Form for Adding/Updating Accomodations

  1. 1.

    Fill out the basic informations of the form.

  2. 2.

    Enter accommodation name in the Name field.

    Enter accommodation description in the Description field.

    Enter accommodation price in the Price field.Note that the price has to be a number

    Enter number of beds in the Beds field.Note that the beds has to be a number

  3. 3.

    To upload a picture for the accomodation , drag and drop the image file into the marked area.

    If the file is uploaded successfuly you will se the preview of the image

    You can add multiple images for the accomodation, by repeating the first step

  4. 4.

    You can remove added image by clicking on it.

  5. 5.

    To finish adding basic information, click on the next button, or if you changed your mind you can click on the cancel button
