Form for Adding/Updating attractions

  1. 1.

    Fill out the basic informations of the form.

  2. 2.

    Enter attraction name in the Name field.

    Enter attraction description in the Description field.

    Enter restaurant price in the Price field.Note that the price has to be a number

  3. 3.

    To upload a picture for the attraction , drag and drop the image file into the marked area.

    If the file is uploaded successfuly you will se the preview of the image

  4. 4.

    To finish adding basic information, click on the next button, or if you changed your mind you can click on the cancel button
