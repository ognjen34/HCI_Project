Form for Adding/Updating Restaurants

  1. 1.

    Fill out the basic informations of the form.

  2. 2.

    Enter restaurant name in the Name field.

    Enter restaurant description in the Description field.

    Enter restaurant rating in the Rating field.Note that the rating has to be a number between 1-5

  3. 3.

    From the drop-down list select Cuisine Type.

  4. 4.

    To upload a picture for the restaurant , drag and drop the image file into the marked area.

    If the file is uploaded successfuly you will se the preview of the image

  5. 5.

    To finish adding basic information, click on the next button, or if you changed your mind you can click on the cancel buton
