Agent Trip Details

Viewing Trips Details

  1. 1.

    After pressing "Details" on previous page, you will be redirected to the Agent Trip Details Documentation.

    On the Agent Trip Details Documentation, you will see additioal informaition about selected trip.

    On the Agent Trip Details Documentation the following information will be displayed:

    • Accomendation name
    • Accomendation description
    • Accomendation pictures
    • Location pinned on map
    • Trip name
    • Drop down menu for months
    • Drop down menu for years
    • How many times trip has been sold
    • Total earnings for this trip
    • List of sold attractions and restaurants
    • Total earnings for each attraction/restaurant
  2. 2.

    To go through pictures of accomendation, press one of the blue arrows below picture of accomendation

    This will switch to next/previous picture in list.

  3. 3.

    To see information about earnings and number of times that certain trip is sold, choose a month/year from drop down menu

  4. 4.

    After pressing on accomendation adress, below map, or attraction/restaurant, pin will show on map.
