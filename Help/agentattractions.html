Agent Attractions

Viewing Attractions

  1. 1.

    After navigating to the Agent Attractions page, you will see a list of attractions available.

    The attraction cards on the Agent Attractions page will display the following information:

    • Attraction Name
    • Address
    • City
    • Price in euros
  2. 2.

    To add a new attraction, click on the Add button.

    This will open an empty form for adding a new attraction.

  3. 3.

    To edit an attraction, click on the Edit button associated with the attraction.

    This will open a form for editing the attraction that the edit button was clicked on.

  4. 4.

    To delete an attraction, click on the Delete button associated with the attraction.

    This will open a confirmation dialog warning you about the consequences of deletion.

    THIS ACTION IS IRREVERSIBLE! Once an attraction is deleted, it cannot be undone!
