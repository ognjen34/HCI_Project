Form for Adding/Updating Trips

  1. 1.

    Fill out the basic informations of the form.

  2. 2.

    Enter trip name in the Name field.

    Enter trip description in the Description field.

  3. 3.

    From the drop-down list select of avaliable accommodations select Accommodation.

  4. 4.

    To upload a picture for the trip , drag and drop the image file into the marked area.

    If the file is uploaded successfuly you will se the preview of the image

  5. 5.

    To finish adding basic information, click on the create button, or if you changed your mind you can click on the cancel button
